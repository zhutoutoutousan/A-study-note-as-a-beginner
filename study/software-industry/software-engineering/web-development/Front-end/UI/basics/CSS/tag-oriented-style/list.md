# Ordered list
## Syntax
```html
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
<ul>
```
```<li>```: list item
# Unordered list
```html
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
<ol>
```
